Introdução ao Grid

Grid é usado para layouts bidimensionais — com linhas E colunas.

⬅ Voltar ao Hub

Conceitos Básicos

O CSS Grid organiza os elementos em linhas e colunas, permitindo criar layouts flexíveis e precisos.

  • display: grid; — ativa o layout de grade.
  • grid-template-columns: define o número de colunas.
  • grid-template-rows: define o número de linhas.
  • gap: define o espaçamento entre os itens.
