Introdução ao Flexbox

Flexbox é usado para layouts unidimensionais — com linhas OU colunas.

⬅ Voltar ao Hub

Conceitos Básicos

O modelo Flexbox permite alinhar, distribuir e ajustar elementos de forma dinâmica dentro de um contêiner.

  • display: flex; — ativa o modo flexível.
  • justify-content: alinha no eixo principal (horizontal).
  • align-items: alinha no eixo cruzado (vertical).
  • flex-direction: define se os itens ficam em linha ou coluna.
